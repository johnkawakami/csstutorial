This is a eight-column wide column. The Boostrap grid is based on 12 columns. You define a column as spanning one or more grid columns by using a class. The classes are named like this: col-md-8. The middle word is the "size" of the device, and determines when a stacked layout "breaks" and becomes a columnar layout. The sizes are: xs, sm, md, and lg. Resize the window to see the grid in action.
